# Useful built-in functions of CPP

### To swap to variables 
```cpp
swap(10, 20);
```

### To check if character is alpha numberic
```cpp
isalnum('?');
```

### To convert character to lower case
```cpp
tolower('A');
```

### To convert character to upper case
```cpp
toupper('b');
```

### To convert integer to string
```cpp
to_string(10);
```
